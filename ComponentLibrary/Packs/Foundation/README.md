# Foundation Pack

## 组件

- `CooldownComponent`
- `LifetimeComponent`
- `StateFlagComponent`
- `ConditionGateComponent`
- `TagComponent`
- `DataBlackboardComponent`
- `TickSchedulerComponent`

## 依赖

- 需要 `ComponentBase`（位于 `ComponentLibrary/Dependencies/component_base.gd`）
- `TickSchedulerComponent` 可选接入 `LocalTimeDomain`

## 使用

- 提供通用基础能力，可被任意玩法层直接复用。
