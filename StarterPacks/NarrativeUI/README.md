# NarrativeUI

叙事 UI 标准玩法包，面向对话驱动和调查驱动的 2D 段落。

- 入口场景：`Main.tscn`
- 平台依赖：`SaveSystem`、`dialogue_manager`、`modify_test`
- 默认内容：开始对话、继续进度、保存/读取、ToastFeed、SaveSlotPanel
- 宿主协议：支持 `start_pack`、`export_pack_state`、`import_pack_state`、`pack_finished`
- 对话资源：`addons/dialogue_manager/modify_test/demo/demo_dialogue.dialogue`
