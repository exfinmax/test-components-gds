# Architecture Decision Records

本目录用于记录本项目的重要架构决策，避免后续重构时反复争论同一问题。

## 状态说明

- `Proposed`：提案中
- `Accepted`：已采纳并执行
- `Deprecated`：已过时
- `Superseded`：被新 ADR 替代
- `Rejected`：已否决
