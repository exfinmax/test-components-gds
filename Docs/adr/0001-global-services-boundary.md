# ADR-0001: 全局服务边界与单一事实来源

## Status

Superseded by ADR-0003

## Context

仓库曾同时存在全局与局部的同类能力实现，导致维护成本高、行为不一致。

## Decision

- 事件、对象池、全局时间必须收敛为单一事实来源。
- 模块层不再随意复制同类基础设施。

## Consequences

- 明确了服务边界，便于后续收敛。
- 该 ADR 的“适配层保留策略”被 ADR-0003 进一步收紧。
